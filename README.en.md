# KK_ExternalVoiceEditor-SBV2

[日本語README](README.md)

** Overview **
- Assistance with editing KK_ExternalVoiceLoader data for using custom voices in KoiKatsu's Character Studio (KK_ExternalVoiceLoader itself is not required)
- Use AI (Style-Bert-VITS2) to synthesize speech from any text and create audio files.

** Install **
1. Place the downloaded Install-KK_ExternalVoiceEditor-SBV2.bat file (Setup-Python.bat is also required) in any folder (folder names must be alphanumeric) and double-click it.
2. Close the window when “Environment setup is complete.” is displayed.
3. Double-click the App.bat file located inside the created KK_ExternalVoiceEditor-SBV2 folder.

![KK_ExternalVoiceEditor-SBV2 KoiKatsu External Voice](https://github.com/user-attachments/assets/45e292c3-972e-4d9e-92b5-2a3ddd50f893)

** How to Use External Voice Files for KoiKatsu **
- Model: Currently unused(Plan to select model information generated by AI speech synthesis)
- Personality: Koikatsu's personality
- Reload: The selected model and personality will be reflected in the voice list.
- Voice list: Click to select
- △▲▼▽: Change the order of the selected voices, up to top, up, down, down to bottom
- Add: Add voice
- Change: Change voice
- Delete: Delete voiceDouble-click the App.bat file located inside the created KK_ExternalVoiceEditor-SBV2 folder.
- Voice: Play selected voice
- Output path: Install voice(wav file, ogg file), plugin(KK_ExternalVoiceLoader.dll, KKS_ExternalVoiceLoader.dll), voice list mod(KK_KKS_custom-voice-list-1.0.0.zipmod) to the specified path
- Output destination: path specification, Koikatsu Party, Koikatsu, Koikatsu sunshine output destination selection(Koikatsu Party, Koikatsu, Koikatsu Sunshine Displayed if them is installed)
- Message: Show execution results
- Start button: Start Koikatsu Party, Koikatsu, Koikatsu sunshine(Koikatsu Party, Koikatsu, Koikatsu Sunshine Displayed if them is installed)

** Precautions **
- Please save the voice you want to use in the UserData folder.（For the input path, specify the folder path after the UserData folder.）
- Please use wav files or ogg files for voices.
- If the output path is a path specification, the plugins (KK_ExternalVoiceLoader.dll, KKS_ExternalVoiceLoader.dll) will not be copied.
- If you want to keep a backup, please copy the app.db file and UserData folder. You can restore by overwriting these two.
- Audio cannot be played unless the path and file name are correct.
- The output destination selection and start button will be displayed if Koikatsu Party, Koikatsu, Koikatsu sunshine is installed.
